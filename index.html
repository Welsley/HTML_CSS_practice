Common CSS Properties

Property Description
color Defines the foreground colour of an element. i.e. the text colour.
background-color Defines the background colour of an element.
font-family Defines the font the element's text uses.
font-weight Defines the weight the element's text uses.
  • normal
  • bold
  • bolder
  • thin
  • thinner
  • etc.
font-size Defines the size the element's text uses. Can be specified as:
  • xx-small
  • x-small
  • small
  • medium
  • large
  • x-large
  • xx-large
  • xxx-large
  • rem
  • px
  • %
  • Can be defined as smaller or larger than the parent element.
font-style Defines the style the element's text uses. Can be specified as:
  • normal
  • italic
  • oblique
  • oblique 10deg (or 2, or 13, or 25 , or 90, etc.)
text-decoration Defines the decoration on text. Can be specified as:
  • none
  • underline
  • overline red
